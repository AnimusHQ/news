# Connectors

This inventory is the connector source of truth for Launch Slice L1 and future
work. Status values are intentionally conservative:

- `Implemented`: code exists and is covered by default verification.
- `Partial`: a safe boundary, dry-run, sidecar contract, or manual lane exists,
  but real production execution is not fully proven.
- `Planned`: documented future connector only.

Each connector record includes: connector_id, category, purpose, lifecycle stage,
current status, execution mode, default enabled, requires network, requires
credentials, requires local binary/model, requires GUI/session, input artifacts,
output artifacts, gates before execution, gates after execution, security risks,
secrets required, audit requirements, failure behavior, and test strategy.
